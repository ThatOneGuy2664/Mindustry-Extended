# What is this game then?
You are the Sharded, a new faction emerging as two different "branches" from Erekir and Serpulo, each one specialized for a different task yet on the same team. Every new planet you go to has a vast variety of resources. New technology must be developed for each of them. Experiment with early iterations of tech, futuristic machines, and so much more. There are currently two full planets, with a third WIP as of now.
